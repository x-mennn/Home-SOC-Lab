# 🛡️ Home SOC Lab

## 📌 Overview

This repository documents my journey of building a complete Home Security Operations Center (SOC) Lab from scratch.

The lab is designed to simulate real-world cyber attacks, collect endpoint logs, detect threats using SIEM, investigate security incidents, and document findings following industry best practices.

---

## 🎯 Objectives

- Build a fully functional Home SOC Lab
- Learn SIEM implementation using Wazuh
- Monitor Windows endpoint logs with Sysmon
- Simulate attacks using Kali Linux
- Detect malicious activities
- Map detections to MITRE ATT&CK
- Create professional incident reports
- Build a cybersecurity portfolio
