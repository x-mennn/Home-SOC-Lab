# 📸 Screenshots

## Overview

This directory contains screenshots captured throughout the development of the Home Security Operations Center (SOC) Lab.

The screenshots provide visual evidence of the installation process, system configuration, attack simulations, detection validation, dashboard monitoring, and incident investigation documented within this repository.

They are organized into subdirectories to improve navigation and simplify the verification of each project phase.
