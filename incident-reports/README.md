# Incident Reports

## Overview

This directory contains SOC-style incident reports created from alerts generated during controlled security validation exercises performed in the Home SOC Lab.

Each report documents the investigation process from initial detection to final analysis using endpoint telemetry collected through Sysmon and alerts generated by Wazuh.

---

## Reports Included

- PowerShell Execution
- Windows Discovery Commands
- Certutil Encode/Decode
- MSHTA Execution
- Regsvr32 Execution

---

## Report Structure

Each report contains:

- Incident Summary
- Detection Details
- Description
- MITRE ATT&CK Mapping
- Investigation Process
- Evidence Collected
- Analysis
- Impact Assessment
- Response Actions
- Lessons Learned

---

## Purpose

The purpose of these reports is to demonstrate SOC investigation workflows, alert triage, detection analysis, and incident documentation skills using validated detections from the Home SOC Lab.

These reports showcase the process of analyzing security alerts using Wazuh SIEM and Sysmon endpoint telemetry.
